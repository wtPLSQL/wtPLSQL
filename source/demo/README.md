### Install Procedure

1) sqlplus SYS/password as SYSDBA @install
2) exit
3) Compare install.LST to installO.LST


### UnInstall Procedure

1) sqlplus SYS/password as SYSDBA @uninstall
2) exit
3) Compare uninstall.LST to uninstallO.LST
